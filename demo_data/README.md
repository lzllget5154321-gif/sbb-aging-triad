# Demo Data — CHARLS-format Simulated Dataset

> **WARNING**: This is entirely synthetic data. NO real participant information.
> **Purpose**: Verify code executability for third-party reviewers.
> **N**: 200 participants, 1 baseline wave (2011)
> **Columns**: 55 harmonized variables

## Data Dictionary

### Demographics
| Column | Type | Description | Real Data Mean (SD) |
|--------|------|-------------|---------------------|
| `id` | int | Simulated participant ID (1-200) | — |
| `wave` | float | Survey wave (all 2011.0 = baseline) | — |
| `age` | float | Age in years | 63.7 (8.5) |
| `sex` | float | 1=male, 2=female | ~48% male |
| `female` | int | Binary female indicator | 52% |
| `education_years` | float | Years of education | ~6 |

## Generation

Data generated by `generate_demo_data.py` using NumPy random distributions (seed=42).
Each variable's distribution parameters were estimated from actual CHARLS baseline data (N=2,639) to preserve realistic ranges, means, and variances.

## Usage

```bash
# Run demo pipeline
python v11_master_pipeline.py --demo

# Output: demo_output/tables/*.json (Step 1-5 results)
```
